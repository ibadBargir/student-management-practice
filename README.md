## 📖 About this Project
Ye ek **practice/demo project** hai jo Spring Boot ke sath banaya gaya hai.  
Is project ka purpose hai **CRUD operations** ko practice karna bina real database use kiye.

---

## 🚀 Features
- 2 types ke service logic implement kiye gaye hain:
  1. **ArrayList based Service** → `StudentServiceArrayList`
  2. **HashMap based Service** → `StudentServiceMap`
- Dono ke liye alag Controllers banaye gaye hain.
- REST API endpoints ke through CRUD operations perform kiye gaye hain.
- Database connect nahi hai, sirf **in-memory (fake database)** use hua hai.

---

## 🛠 Tech Stack
- **Java 17**
- **Spring Boot 3.x**
- IDE: IntelliJ IDEA / Visual Studio Code

---

## 📌 API Endpoints (ArrayList Service Example)
- `GET /students` → Get all students
- `GET /students/{id}` → Get student by id
- `POST /students` → Add new student
- `PUT /students/{id}` → Update student
- `DELETE /students/{id}` → Delete student
